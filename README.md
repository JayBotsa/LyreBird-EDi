   Example: `D7:1972` → Dam 7 is exactly 1972 mm deep. Nothing more. Nothing less.
   Example: `D6:1` → Pump 1 is exactly on. Nothing more. Nothing less.

3. **Mirror** – your device (Grok instance, phone, robot).
